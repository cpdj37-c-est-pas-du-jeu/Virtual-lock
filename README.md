Elements :
- Compte a rebour
- Si le compte a rebour arrive a son echeance, echec.
- Si bonne saisie du mot-de-passe, arret du compte a rebour et affichage d'une image/video

Fonctionnalites additionnels :
- Configuration via formulaire

Idees d'ajouts :
